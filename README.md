## About The Project
## ANALISIS PENGHITUNGAN TOTAL PENJUMLAHAN ELEMEN DALAM SEBUAH ARRAY
Program ini berfungsi menghitung total (sum) dari input array integer. Pengguna cukup memasukkan jumlah elemen (n) dan nilai-nilainya, kemudian program akan memberikan output berupa hasil penjumlahan serta durasi waktu yang dibutuhkan untuk menyelesaikan perhitungan
